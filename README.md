# Smart Parking Management System

## Description
Smart Parking Management System is a web-based application that helps manage parking slots efficiently. 
The system allows an admin to view available parking slots, allocate parking to vehicles, search vehicle details, and track parking activity. 
It also provides features like automatic slot suggestion, earnings calculation, and a simple dashboard interface.

## Features
- Admin login system
- Parking slot visualization
- Vehicle number search
- Automatic slot suggestion
- Earnings tracking
- Dark mode interface
- Parking analytics

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Run the `login.html` or `index.html` file in your browser.
